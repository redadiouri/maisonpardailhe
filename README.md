1. Installez les dépendances nécessaires :
    ```bash
    npm install
    ```
    ou
    ```bash
    yarn install
    ```
2. Créez un fichier `.env` à la racine du projet et renseignez les variables d'environnement requises.
3. Lancez le serveur de développement :
    ```bash
    npm run dev
    ```
    ou
    ```bash
    yarn dev
    ```
